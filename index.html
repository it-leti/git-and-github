Git & GitHub

Для всех и для каждого

С помощью этого руководства Вы овладеете Git — невероятно удобной и мощной системой управления версиями, а также познакомитесь с GitHub — крупнейшим сервисом для хранения кода и совместной разработки.

Так как детальное изучение Git подразумевает обращение с командной строкой, настоятельно рекомендуем предварительно в этом попрактиковаться.

От Вас также потребуется желание разбираться, проводя эксперименты и проверяя идеи. Только практикуясь самостоятельно, Вы сможете чему-то научиться.
